# django-simple_movie
用django做一个简单的电影网站<br>
# 描述
新手上路，拿来练练手，功能比较简单，代码也不复杂。因为没有片源，暂时没有添加播放功能。
# 开发环境
win10<br>
pycharm<br>
django 3.1.6<br>
python 3.9.1<br>
没有过多的依赖包
